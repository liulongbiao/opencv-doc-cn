# OpenCV 教程

以下链接描述了一系列基础 OpenCV 教材。
所有的提到的代码都作为 OpenCV 常规发布版一起提供，因此你可以在复制粘贴前查看一下。
以下教程自动由我们 GIT 仓库里的 reST 文件自动生成。

我们总是欢迎听到你的评论以及接收你对任何教程的贡献。
